Course Description

Almost all electrical engineering technologies or systems which make decisions rely on a measurement system one way or the other. Something has to be measured for a decision to be made. Of recent there has been a hype on Internet of Things (IoT), data science, Industrial Internet of Things (IIT), etc. These technologies are highly dependent on data. This data is measured at particular areas of interest and thus these technologies strongly depend on the integrity of the measurement systems. They are as good as the measurement system employed. This course is concerned with the instrumentation systems concepts and terminology. Static and dynamic properties of measurement systems, calibration. Introduction to statistical techniques for handling measurement uncertainty. Noise. Transducers based on varying R, L, C. Signal conditioning. Thermocouples, piezoelectric transducers.

If two or more measurement applications are submitted for approval the group which submitted first will be given that measurement application to work as their project.
